h1. dcdj - Dans combien de jour

dcdj est une application pour calculer le nombre de jour entre une date et maintenant basé sur JQuery Mobile et RoR3.

h2. Introduction

Ayant un enfant qui adore demander pour chaque évenement "c'est dans combien de jours", et cela pour différents événements et tous les jours, j'ai développé rapidement dcdj pour pouvoir lui répondre simplement en consultant mon iPhone.

Cette application est donc optimisé pour téléphone mobile via l'utilisation du framework "JQuery Mobile":http://jquerymobile.com/ pour le frontend et s'appuie sur le grand classique "Ruby on Rails":http://rubyonrails.org/ pour la beckend.
